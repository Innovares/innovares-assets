# Innovares Public Assets

Este repositorio funciona como una CDN (Content Delivery Network) centralizada para los activos digitales públicos de **Innovares**.

## Contenido
Aloja archivos estáticos destinados a la distribución pública y enlaces externos, tales como:
- **Identidad de Marca:** Logos oficiales (SVG, PNG), iconos y favicons.
- **Materiales de Marketing:** Banners, gráficos para redes sociales e imágenes promocionales.
- **Recursos para Correo:** Imágenes y activos requeridos para las plantillas de correo y firmas oficiales.

## Uso
Estos archivos son accesibles públicamente a través de sus URLs directas (raw). Están destinados para su uso en:
- Sitios web y aplicaciones oficiales de Innovares.
- Comunicaciones por correo electrónico de la empresa.
- Integraciones con socios autorizados.
